# SorHub PvP - Blox Fruits PvP Script

## 🎯 Возможности:

### ⚔️ AIMBOT (НАСТРАИВАЕМЫЙ)
- Full Screen Mode (аимбот на весь экран)
- Radius Mode (аимбот в радиусе)
- Visible Check (проверка видимости)
- Team Check (не целиться в своих)
- Prediction (предсказание движения)
- Выбор части тела (Head/Torso)

- Автоматический поиск игроков
- Приоритет: низкое здоровье
- Игнорирование своей команды
- Safe Mode (защита от смерти)
- Auto Respawn

### 👾 BOSS FARM
- Автоубийство боссов
- Сбор дропа
- Выбор приоритета

### 🛡️ АНТИ-БАН
- Случайные прыжки
- Имитация движений мыши
- Анти-афк система
- Защита от детекции
